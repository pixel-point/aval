# @pixel-point/aval-format

Strict parser, validator, writer, H.264/H.265/VP9/AV1 inspectors, PNG decoder,
and graph adapter for compiled AVAL wire format `1.1`. Hostile byte arithmetic,
layout, media-profile, and digest gates remain checked without imposing a
small-asset product ceiling.

See the repository [format `1.1` reference](../../docs/format/1.1.md) and
[authoring guide](../../docs/compiler/authoring-video-and-states.md).
