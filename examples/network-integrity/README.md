# Network and integrity illustration

This source example is intentionally illustrative. The `.example.test` origin,
codec bundle, consumer-owned alternate image, MIME strings, and integrity tokens are
placeholders; it must not be described as a live endpoint. Replace every
source with the matching immutable asset URL plus the exact `type` and
`integrity` from `build.json` before running `npm install` and `npm run build`.

For an immediately runnable local workflow, use `avl init` and `npm run dev` in
the generated starter.
